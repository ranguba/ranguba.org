Module: Groonga::Pagination

Defined in:
lib/groonga/pagination.rb

Overview

ページネーション機能を追加するモジュール。

ページ番号やレコードが何番目かは0ベースではなく1ベースで あることに注意すること。

def end_offset
  return nil if @n_records.zero?
  [start_offset + @page_size - 1, @n_records].min
end
